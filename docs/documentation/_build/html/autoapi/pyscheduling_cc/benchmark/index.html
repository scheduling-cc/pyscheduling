pyscheduling_cc.benchmark.run_solver_instance(fname: pathlib.Path, instances: list[pyscheduling_cc.Problem.Instance], methods_args: list[object], kwargs={}, log_param: list[Log] = [Log.objective, Log.runtime])#

Performs the benchmark of a list of methods on a list of instances, creates a csv file of the results and outputs the results

Parameters

  • fname (Path) – output filename

  • instances (list[Problem.Instance]) – List of instances to solve

  • methods_args (list[object]) – List of either methods or couple of method and its extra-arguments.

  • kwargs (dict, optional) – Extra-arguments for all methods. Defaults to {}.

  • log_param (list[Log], optional) – parameters to be logged. Defaults to [Log.objective,Log.runtime].

Returns

result of the benchmark

Return type

dict
